## Whisper (Local)

This option installs whisper-rust to allow all speech to text to be done locally on device.

```bash
# Set a local STT service
01 --stt-service local-whisper
```

## Whisper (Hosted)

```bash
# Set STT service
01 --stt-service openai
```
